## Question #1: Given the following HTML page, how can we make the paragraphs inside the `main` container have font size 32px?
``` html
<!DOCTYPE html>
<html>
  <head>
  </head>
  <body>
    <header>
      <h1>ACME</h1>
      <p>ACME is a company...
      </p>
    </header>
    <main>
      Some words about ACME current status:
      <p>
        ACME value has dramatically decreased...
      </p>
    </main>
  </body>
</html>
```

 
*Multiple Choice*

(1)
```
main p {
  font-size: 32px;
}
```

(2)
 
```
p {
  font-size: 32px;
}
```

(3)
```
body {
  font-size: 32px;
}
```

(4)
 
```
html {
  font-size: 32px;
}
```
